## 개요
운영체제에 대해 공부하다가 Peer-to-peer에 대해 알게 되었습니다. 주 1회 블로그 글쓰기 하는 스터디원들끼리 매주 일요일에 모여서 작성한 블로그 글에 대해 이야기합니다. 그러던 중, peer-to-peer가 우리가 모인 디스코드와 유사한 것 같아서 흥미가 생겨 해당 프로젝트를 시작했습니다.
